C#基础

输入输出

C#的输入和输出就四个函数：

数据类型

类型 描述
char 16位Unicode字符
decimal 128位十进制数字
long 64位有符号数

引用类型

引用类型不包含存储在变量中的数据类型，但是包含对变量的引用。引用类型有三种，对象object，动态dynamic，字符串string。

对象类型

对象类型object，是所有数据类型基类，对象类型可以被分配成其他类型。在分配之前需要进行类型转换。

将值类型转化为引用类型，称为装箱，将引用类型转换值类型，称为拆箱。

对象的类型检查是发生在编译期。

动态类型

动态类型是可以存储任何类型在动态类型变量中，但是这些变量的类型检查是发生在运行时。

字符串类型

字符串类型可以带@符号，也可以不带@符号。不带@符号，与我们通常使用的字符串没有区别。如果带@符号，则表示忽略掉转义字符的作用。

类型转换

类型转换，一般分为两种，一种是隐式转换，隐式转换一般是小类型转化为大类型，不会造成数据损失。显示转换即强制类型转换会造成数据损失。也可以使用Convert库来进行数据类型转换。

foreach循环

internal修饰符

被internal修饰的变量或者函数，可以被当前的程序调用。

参数传递

参数传递的方式有三种：按值传递，按引用传递，按输出传递。

按值传递跟C++区别不大，不再赘述。

按引用传递

按引用传递，实参和形参前面都需要加上ref关键字。

数组

C#的数组在声明上和C++还是有区别的。

多维数组

数组参数

params关键字

当一个函数中含有params关键字的时候，那么这个函数就是变参函数。此函数会将传递的参数打包到数组中。

  • 使用 params关键字可以指定一个方法参数,该方法参数的数目可变。

  • 可以发送参数声明中所指定类型的逗号分隔的参数列表或指定类型的参数数组。 还可以不发送参数。 如果未发送任何参数，则 params 列表的长度为零。

  • 在方法声明中的 params 关键字之后不允许任何其他参数，并且在方法声明中只允许一个 params 关键字。

  • 声明的 params 参数类型必须是一维数组

字符串string

  • 通过检索属性或调用一个返回字符串的方法

结构体

C#的结构体和C++的结构体类似，但是仍然有不同的地方：

  • 结构体可以定义构造函数，但是不能定义析构函数，而且不能定义默认的构造函数。
  • 结构体不能继承
  • 结构体不能指定virtual，abstract，protect。
  • 结构体默认属性为私有。
  • 结构体可以不使用new来进行构造，但是需要注意的是如果不使用new来进行构造，那么必须先将结构的成员进行赋值之后，才能使用。

枚举

枚举和C++类似，只是赋值的时候，要用.运算符来指定枚举类型。
